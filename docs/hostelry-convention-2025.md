## 📐 Cálculo del Coste/Hora para la Empresa

El **coste por hora para la empresa** NO es solo el salario bruto dividido entre horas. Incluye:

### Componentes del Coste Laboral

1. **Salario Base Bruto** (según convenio)
2. **Prorrateo de Pagas Extra** (2 al año)
3. **Vacaciones** (30 días naturales)
4. **Festivos y Días de Libre Disposición**
5. **Contingencias Comunes** (Seguridad Social)
   - Cuotas empresariales SS: ~23,60%
   - FOGASA: 0,20%
   - Formación: 0,70%
   - Desempleo (empleador): 5,50%

### Fórmula Simplificada

```
Coste/Hora Empresa = (Salario Bruto Anual / 1.736 horas) × 1,40

Donde:
- 1.736 horas = jornada anual completa (40h/sem × 52 sem - vacaciones - festivos)
- 1,40 = factor que incluye SS y demás cargas (~40% adicional)
```

### Ejemplo Práctico: Cocinero

| Concepto | Valor |
|----------|-------|
| Salario Base Mensual | 1.500,00 € |
| Pagas Extra (2/14) | 214,29 € |
| Salario Mensual Prorrateado | 1.714,29 € |
| Salario Anual Bruto | 20.571,48 € |
| + Seguridad Social (23,6%) | 4.854,87 € |
| + Otros Costes Laborales | 3.085,74 € |
| **COSTE ANUAL TOTAL** | **28.512,09 €** |
| **Coste/Hora Empresa** | **16,42 €/h** |

## 📝 Notas Importantes

### Variaciones por Comunidad Autónoma

- Algunas CC.AA. tienen convenios propios con salarios superiores
- **País Vasco**: +10-15%
- **Cataluña**: +8-12%
- **Madrid**: +5-10%
- **Baleares**: +12-18% ( turismo)

### Otros Factores a Considerar

1. **Antigüedad**: Muchos convenios establecen trienios o quinquenios
2. **Turnicidad**: Plus de disponibilidad y trabajo en festivos
3. **Horario Partido vs. Continuo**: Diferencias en compensación
4. **Categoría del Establecimiento**: Hotel 5★ vs. Bar de barrio

### Actualización Anual

El SMI se actualiza cada año (normalmente en marzo). Revisa esta documentación anualmente para ajustar los valores.

## 💡 Consejo Práctico

**Para calcular el coste real de tu empleado:**

1. **Empieza con el salario base del convenio**
2. **Añade un 40% para Seguridad Social y cargas**
3. **Considera +5-10% por antigüedad media**
4. **Añade plus de disponibilidad si procede**
5. **Redondea al alza para margen de seguridad**

**Ejemplo con margen de seguridad:**
- Cocinero (convenio): 1.500€/mes
- +40% SS: 2.100€/mes
- +5% antigüedad: 2.205€/mes
- **Coste/hora recomendado**: ~16,50 €/h
