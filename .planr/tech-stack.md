# Tech-Stack: Restaurant Finance App

## Core Framework
- **Next.js 16.1.6**: React framework with App Router for server-side rendering and server actions.
- **React 19.2.3**: Component-based UI library.
- **TypeScript 5**: Static typing for enhanced developer experience and error prevention.

## Styling & UI
- **Tailwind CSS 4**: Utility-first CSS framework for rapid and consistent styling.
- **Framer Motion 12**: Animation library for smooth transitions and interactive feedback.
- **Lucide React**: Vector icons library.
- **Radix UI**: Primitive components for accessible and premium UI patterns.

## Data & Backend
- **Supabase**: Backend-as-a-Service (BaaS) providing PostgreSQL, Auth, and Storage.
- **Zod**: TypeScript-first schema declaration and validation.
- **Date-fns**: Modern JavaScript date utility library.

## Testing & Quality Assurance
- **Vitest**: Fast unit and integration testing framework.
- **Shortest (AI)**: Natural Language Processing (NLP) based E2E testing using Claude.
- **EvoMaster (AI)**: Evolutionary and dynamic analysis for API fuzzing and test case generation.
- **Testsigma (AI)**: Agentic test automation platform for cross-platform regression.
- **Playwright**: Low-level browser automation engine.

## DevOps & Tools
- **GitHub Actions**: CI/CD automation.
- **Docker**: For running EvoMaster and local development environments.
- **Pino**: High-performance JSON logging for structured observability.
